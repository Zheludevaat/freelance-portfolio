# Neighbourhood Café Landing Page

A modern, mobile-first single-page website for a neighbourhood café featuring:
- Responsive hero section with call-to-action
- Interactive menu gallery
- Location map integration
- Contact form with validation

## How to Use

1. Download all files
2. Open `index.html` in any web browser
3. The site works offline and requires no server

All code is self-contained with no external dependencies except for the map service.
